# Documentation Directory

This directory contains all documentation for the Migration Analysis System.

## Directory Structure

### `/setup/` - Setup Documentation
Installation and configuration guides:
- **WINDOWS_SETUP.md** - Windows-specific setup instructions
- **SETUP_SUMMARY.md** - Quick setup summary
- **README_SETUP.md** - Detailed setup guide

### `/guides/` - User Guides
How-to guides and tutorials:
- **QUICK_START.md** - Quick start guide for new users
- **NEW_COMMANDS.md** - Documentation of available commands
- **TASKS_3_4_GUIDE.md** - Guide for Tasks 3 & 4
- **examples.md** - Usage examples

### `/fixes/` - Fix Documentation
Technical summaries of bug fixes and improvements:
- **API_EXTRACTION_FIX_SUMMARY.md** - Vue.js API call extraction fix
- **PHP_TABLE_EXTRACTION_FIX_SUMMARY.md** - PHP table extraction fix

### Root Documentation
- **IMPLEMENTATION_SUMMARY.md** - Overall implementation summary
